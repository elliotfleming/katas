# Tennis Game

## Rules

    0 = Love
    1 = Fifteen
    2 = Thirty
    3 = Forty
    4 = Win

    1-0 => Fifteen-Love
    2-0 => Thirty-Love
    3-0 => Forty-Love
    4-0 => Winner

    0-0 => Love-All
    1-1 => Fifteen-All
    2-2 => Thirty-All
    3-3 => Deuce
    4-4 => Deuce

    Must win by 2 points.
    More concerned with position than points.

    4-3 => Advantage Player 1
    5-3 => Winner
