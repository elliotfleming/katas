# Roman Numerals

## Rules

You must accept any positive number through 5000 and return the proper Roman numeral for that number. The rules for converting a number are as follows:  

* 1    => 'I'
* 4    => 'IV'
* 5    => 'V'
* 9    => 'IX'
* 10   => 'X'
* 40   => 'XL'
* 50   => 'L'
* 90   => 'XC'
* 100  => 'C'
* 400  => 'CD'
* 500  => 'D'
* 900  => 'CM'
* 1000 => 'M'
